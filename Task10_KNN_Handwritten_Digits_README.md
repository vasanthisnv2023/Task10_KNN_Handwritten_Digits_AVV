# Task 10 – Handwritten Digit Classification using KNN

## Objective
Classify handwritten digits using the K-Nearest Neighbors algorithm.

## Dataset
Sklearn Digits Dataset (8×8 images).

## Steps
- Data loading and visualization
- Feature scaling
- KNN model training
- K value tuning
- Accuracy vs K analysis
- Confusion matrix evaluation

## Outcome
The model successfully classifies handwritten digits and demonstrates the importance of K selection in KNN.
